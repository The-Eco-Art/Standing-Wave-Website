## Project Structure

This is a lightweight, dependency-free static site designed for performance and longevity.

- `index.html`: The main entry point containing the core framework logic and structure.
- `theme.css`: Centralized color variables and theme definitions (Dark, Light, Warm, Ocean).
- `wavecss.css`: Core layout, responsive design, and animations.
- `css/slides.css`: Styling for individual slide components.
- `js/`: Vanilla JavaScript modules for carousel navigation and wave visualization.
- `slides/`: HTML templates for the slide content (Extrinsic, Node, Intrinsic).

## Local Development

No build steps are required. To run the site locally:

1.  Clone the repository.
2.  Open `index.html` in any modern web browser.
3.  (Optional) Use a simple HTTP server for better performance:
    ```bash
    python -m http.server 8000
    # Open http://localhost:8000
    ```
